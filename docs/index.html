Echoing the educator, not replacing the magic.

A free, fully offline AI assistant for teachers — lesson plans, feedback, IEP drafts, and more, generated by a model that runs on your own computer. Nothing you type ever leaves the machine.

Every major classroom AI tool asks you to trust a cloud vendor with student data. Local Ed AI structurally can't leak it — the model runs on your machine, through Ollama, with no server in between. That matters most for the highest-stakes student information, like IEP and 504 documentation.

What it does

A prompt library and chat interface built for a teacher's actual workflow — pick a task, add context, get an editable draft.

Prompt library

Ready-to-edit prompts for lesson planning, assessment, feedback, communication, and differentiation.

Ask DittoEd

A general assistant for when you're not sure which tool to reach for — describe what you need.

IEP Form Assistant

Walk through a Maryland IEP section by section, draft narrative fields from your notes, and export a .docx/PDF — always flagged as requiring full IEP team review.

Persistent history

Every chat and IEP draft autosaves locally and can be resumed later with full context — pin the ones you use often.

Projects & search

Group related chats into projects, and full-text search across everything you've saved.

Reusable files

Attach a document once — it's saved locally so you can reuse it in a future chat without re-uploading.

How it works

No account, no subscription, no data plan. Setup is guided the first time you launch.

  1. Download and run the installer for your OS.
  2. First launch checks for Ollama — if you don't have it, the app offers to install it for you.
  3. Pick a starter model sized to your hardware and download it once.
  4. Start prompting — everything after this runs 100% on your machine.
