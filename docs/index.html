How it works

No account, no subscription, no data plan. Setup is guided the first time you launch.

  1. Download and run the installer for your OS.
  2. Click through the SmartScreen warning. The installer is unsigned, so Windows will flag it as from an "unknown publisher." That's expected — click "More info" → "Run anyway."
  3. Let it set up Ollama — the app detects whether you have it and offers to install it for you.
  4. Pick a starter model sized to your hardware, then start prompting. Everything from here runs 100% on your machine.

Support this project

Local Ed AI is a solo-built, open-source project. Development is free; a few real costs stand between "works on my machine" and something a district can hand to every teacher without a second thought:

  • Code signing — an unsigned installer trips Windows SmartScreen and macOS Gatekeeper warnings that scare off exactly the risk-averse school IT departments this project needs to win over. A code-signing certificate removes that warning.
  • An independent domain — pointing a real domain at this same free GitHub Pages site, so the project has a stable, memorable address independent of any one platform.
  • Contributor time — funding to bring on help for the parts one person can't sustain alone: testing across districts, accessibility review, and features requested by actual teachers.
